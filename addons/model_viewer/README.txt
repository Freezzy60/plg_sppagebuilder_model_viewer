# SP Page Builder 3D Model Addon

## Beschreibung
Das SP Page Builder 3D Model Addon ist eine Erweiterung, die es ermöglicht, 3D-Modelle im glb-Dateiformat in SP Page Builder-Seiten anzuzeigen.

## Installation
1. Lade die neueste Version des Addons von [GitHub](https://github.com/Freezzy60/sp-page-builder-3d-model-addon) herunter.
2. Gehe zu deinem Joomla-Administratorbereich.
3. Navigiere zu "Erweiterungen" > "Verwalten" > "Installieren".
4. Wähle die heruntergeladene Datei des Addons (ZIP-Datei) aus und klicke auf "Hochladen und Installieren".

## Verwendung
1. Nach erfolgreicher Installation erscheint ein neues Element namens "Model Viewer" in der Liste der verfügbaren Addons in SP Page Builder.
2. Ziehe das "Model Viewer"-Element in den gewünschten Abschnitt deiner Seite.
3. Klicke auf das Element, um die Einstellungen zu öffnen.
4. Lade deine glb-Datei hoch und stelle die gewünschten Optionen wie Größe, Position und Animation ein.

## Unterstützte Dateiformate
Dieses Addon unterstützt das glb-Dateiformat für 3D-Modelle.
